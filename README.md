# Приложение для управления базой данных
# 📊 Клиентская база (WinForms + Access)

Это desktop-приложение на **C# (Windows Forms)** для работы с базой данных клиентов.  
Позволяет добавлять, редактировать, удалять и фильтровать записи в базе данных **Microsoft Access (.mdb)**.

---

## 🚀 Возможности

- ➕ Добавление новых клиентов  
- ✏️ Редактирование существующих записей  
- ❌ Удаление клиентов  
- 🔍 Фильтрация данных по любому столбцу  
- 💾 Сохранение изменений в базе данных  
- 🔄 Обновление данных из БД  
- 🕒 Отображение текущей даты и времени  
- ✅ Валидация вводимых данных  

---

## 🗂 Структура данных

Таблица **`клиенты`** содержит следующие поля:

| Код | Имя | Фамилия | Пол | Возраст | Телефон | Email | Дата_добавления |
|-----|-----|---------|-----|---------|---------|-------|-----------------|


---

## 🧰 Используемые технологии

- C# (.NET Framework)
- Windows Forms
- ADO.NET
- OleDb (для работы с Access)
- Microsoft Access (.mdb)

---

## 📖 Руководство пользователя

### 🔹 Добавление клиента

1. Заполните поля:
   - Имя
   - Фамилия
   - Пол
   - Возраст
   - Телефон
   - Email
2. Нажмите кнопку **"Добавить"**

📌 Дата добавляется автоматически.

---

### 🔹 Редактирование клиента

1. Выберите строку в таблице
2. Измените нужные поля
3. Нажмите кнопку **"Изменить"**

---

### 🔹 Удаление клиента

1. Выберите строку в таблице
2. Нажмите кнопку **"Удалить"**  
   *(или клавишу Delete)*

---

### 🔹 Сохранение изменений

После добавления/изменения/удаления:

- Нажмите кнопку **"Сохранить"**, чтобы записать изменения в базу данных

---

### 🔹 Обновление данных

- Кнопка **"Обновить"** (с круговой стрелкой) перезагружает данные из базы

---

### 🔹 Очистка полей

- Кнопка **"Очистить"** сбрасывает все поля ввода

---

### 🔹 Фильтрация данных

1. Выберите поле в выпадающем списке
2. Введите значение в поле фильтра

📌 Особенности:
- Для поля **"Код"** используется точное совпадение  
- Для остальных полей — поиск по вхождению (LIKE)

---

### 🔹 Автозаполнение полей

- При выборе строки в таблице данные автоматически подставляются в поля редактирования

---

### ⚠️ Важно

- Все поля обязательны для заполнения
- Телефон должен быть в формате: `+79XXXXXXXXX`
- Email должен быть корректным
- Телефон и Email должны быть уникальными
